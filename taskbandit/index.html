What it does

TaskBandit is a self-hosted household task manager with chore assignment, approvals, recurring work, follow-up tasks, proof photos, live updates, and both browser/PWA and native Android clients.

Templates are organised as group, type, and optional sub-type so daily chore lists stay readable while admins can still model richer routines such as laundry chains, room cleaning, and waste collection.

Core areas

  • Quick Start covers the fastest path to a running local stack.
  • Deployment explains the server and web container split.
  • Reverse Proxy covers HTTPS, CORS, headers, and WebSockets.
  • Clients compares the admin UI, browser/PWA client, and Android app.

Start here

For a new install, begin with the quick start, then review configuration and reverse proxy setup before exposing the app publicly. During first setup, you can optionally import the built-in starter templates with translated labels and linked follow-up tasks.
