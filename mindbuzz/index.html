What it does

MindBuzz is a self-hosted quiz platform for small events, classrooms, team sessions, and training. It builds on the original Rahoot foundation and expands the manager experience with richer quiz administration, run history, exports, settings, media support, and mobile reconnect improvements.

Core features

  • Self-hosted multiplayer quiz sessions with a manager interface and player room-code join flow.
  • Browser-based quiz creation, editing, deletion, launch, and run history.
  • SQLite-backed manager accounts with admin and manager roles.
  • Optional OpenID Connect / OAuth2 SSO for manager sign-in.
  • CSV result exports and support for image, video, remote audio, and uploaded local audio.
