What it does

VATSIM Monitor is a self-hosted platform for tracking controller availability on the VATSIM network. It is designed for users who want local accounts, watched positions, live status, and notifications without requiring VATSIM OAuth.

Core features

  • Live polling of VATSIM controller data.
  • Watch-rule monitoring for specific positions or wildcard patterns.
  • Optional top-down matching for broader position coverage.
  • Discord webhook notifications with per-event customization.
  • Local multi-user accounts, rotating server logs, and a signed-in troubleshooting logs page.

Typical flow

  1. Create a local account.
  2. Add watch rules such as EGLL_TWR or EGNX_%.
  3. Connect one or more Discord webhook channels.
  4. Customize online, offline, and controller-change templates if needed.
  5. Use the monitor UI and receive alerts as staffing changes occur.
